# Async Runtime for Python

Учебный проект, реализующий лёгкий асинхронный runtime на Python.

## О проекте

В проекте создаётся собственный асинхронный движок с событийным циклом и поддержкой будущих результатов.
Он показывает, как работает базовый `EventLoop`, управление задачами и работа с обратными вызовами по завершении операций.

## Основные компоненты

- `async_runtime.event_loop.EventLoop` — простой событийный цикл с очередью задач и обработкой ввода/вывода через `select`.
- `async_runtime.future.Future` — объект, представляющий обещанный результат, с установкой результата, исключения, отменой и регистрацией колбэков.

## Структура

- `async_runtime/` — исходный код runtime
- `tests/` — unit-тесты для проверки поведения событийного цикла и Future
- `pytest.ini` — конфигурация для запуска тестов

## Запуск тестов

```bash
pytest
```

## Пример использования

```python
from async_runtime import EventLoop, Future

loop = EventLoop()
fut = Future(loop)

fut.add_done_callback(lambda f: print('done:', f.result()))
loop.call_soon(fut.set_result, 42)
loop._run_once()
```

## Примечание

Проект демонстрирует основные принципы построения async runtime и может расширяться дальше по мере развития.
